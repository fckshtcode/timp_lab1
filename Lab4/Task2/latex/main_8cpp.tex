Программа выполняет следующие действия\+:
\begin{DoxyEnumerate}
\item Запрашивает количество столбцов для таблицы
\item Создает объект шифра \mbox{\hyperlink{classTable}{Table}} с указанным количеством столбцов
\item Предоставляет меню для выбора операций (шифрование, расшифровка, выход)
\item Обрабатывает ввод пользователя и выводит результаты
\end{DoxyEnumerate}
