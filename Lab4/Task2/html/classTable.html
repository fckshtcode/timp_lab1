Подробное описание

Класс для шифрования и расшифровки текста табличным методом

Реализует шифр маршрутной перестановки, где текст записывается в таблицу по строкам, а считывается по столбцам (или наоборот).

Заметки
Шифрование: текст записывается в таблицу по строкам, считывается по столбцам
Расшифровка: текст записывается в таблицу по столбцам, считывается по строкам

Алгоритм расшифровки:

  5. Считывание текста из таблицы по строкам (слева направо)
  6. Удаление символов-заполнителей '_' из результата
