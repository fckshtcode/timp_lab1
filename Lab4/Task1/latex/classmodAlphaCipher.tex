\doxysubsection{Подробное описание}
Класс для шифрования и расшифровки текста модифицированным алфавитным шифром 

Реализует шифр, использующий русский алфавит и ключ для преобразования текста. Шифр работает по принципу сложения/вычитания числовых значений символов.
